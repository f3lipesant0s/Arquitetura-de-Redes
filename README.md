# Arquitetura-de-Redes
Este repositório é um guia de estudos prático e teórico sobre Arquitetura de Redes de Computadores. O objetivo é documentar minha evolução no entendimento de como os dados trafegam pelo mundo, desde a fiação física até as camadas complexas de segurança e filtragem de pacotes.
Estudando Arquitetura de Redes através do curso de Gabriel Torres na Udemy. Acredito que uma base sólida em redes é o primeiro passo para minha especialização em Cybersecurity. Ao concluir, pretendo validar este conhecimento com uma certificação técnica.
